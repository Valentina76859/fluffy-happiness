  1.2 Моя версия ТЗ 
1.Пользователь может добавить в корзину от 1 до 10 единиц одного товара
2.Для удаления полностью товара из корзины используется отдельная кнопка «Удалить», находящаяся в строке товара
3.В корзине может находится не более 5 уникальных товарных позиций(артикулов)
4.Суммарное количество всех товаров в корзине не может превышать 50 штук 
5. При попытке добавить товар, превышающий лимит, система блокирует действие и показывает сообщение с конкретным нарушением: “Лимит уникальных товаров превышен””Лимит количества конкретных товаров превышен”"Лимит корзины превышен" 
6.На странице корзины отображается список товаров, их количество, цена за единицу и общая стоимость позиции
7. Если пользователь уменьшает количество товара до 0, товар удаляется из корзины автоматически.
8.В корзине отображается реклама других товаров, похожих на то что пользователь недавно смотрел
9.Реклама товаров в корзине должна быть каждый будний день: утром с 8:00 до 12:00 и вечером с 17:00 до 21:00 по местному времени пользователя,
10.Если цена на товар изменилась в каталоге, система должна автоматически обновить ее в корзине у всех пользователей и выделить красным цветом
